# 김도연 데이터 분석가 신입 자기소개서

## 1. [지원동기 및 입사 후 포부] 주도적으로 문제를 정의하고 가치를 창출하는 여정의 시작
저는 단순한 현상 유지를 넘어, 스스로 문제를 정의하고 데이터를 통해 세상에 없던 '가치'를 창출해 내는 과정에 강한 몰입감을 느낍니다. 대학교 재학 시절 수강한 데이터 전공 수업은 제가 데이터 분석이라는 분야에 깊이 매료되는 결정적인 계기가 되었습니다. 정제되지 않은 방대한 데이터 속에서 유의미한 패턴을 발굴하고, 이를 객관적인 의사결정의 지표로 탈바꿈시키는 과정에서 깊은 지적 희열을 느꼈습니다. 누군가 시켜서 하는 수동적인 분석이 아닌, 비즈니스의 페인 포인트(Pain Point)를 선제적으로 찾아내고 데이터로 해결책을 제시하는 '자기 주도적 데이터 분석가'의 역할은 제가 가장 열정을 쏟을 수 있는 확고한 목표가 되었습니다.
이러한 목표를 달성하기 위해 저는 끊임없이 저 자신을 채찍질하며 실무 역량을 다져왔습니다. Python, SQL, Java 등 데이터 전처리와 분석에 필수적인 핵심 기술을 습득한 것은 물론, 데이터에 대한 폭넓은 이해와 전문성을 증명하기 위해 ERP 2급, ADsP, SQLD를 우선적으로 취득했습니다. 현재는 졸업 유예 기간을 전략적으로 활용하여 데이터 분석 역량의 최고봉이라 할 수 있는 빅데이터분석기사 자격증을 취득할 예정입니다. 입사 후에는 제가 가진 탄탄한 분석 역량과 끝없는 탐구 정신을 바탕으로, 조직이 직면한 복잡한 문제를 데이터 기반으로 명쾌하게 해결해 내겠습니다. 언제나 주도적으로 사고하고 먼저 행동하여 조직과 비즈니스의 성장을 견인하는 핵심 크루(Crew)로 자리매김하겠습니다.

## 2. [직무 경험 및 역량] 전처리부터 백엔드 파이프라인까지, 풀스택 데이터 분석 역량
데이터 분석의 핵심은 잘 정제된 데이터를 어떻게 수집하고 통신하느냐에 달려있다고 확신합니다. 저는 단순 분석을 넘어 데이터가 흐르는 '인프라'를 이해하는 분석가로 성장해 왔습니다.
**[데이터 분석 및 전처리 고도화]** 데이터 분석 공모전에 참가해 로우 데이터 정제 과정을 주도하여 **우수상**을 수상했습니다. Pandas, Numpy를 활용해 이상치를 탐지하고 전처리 시간을 [30%] 단축했으며, Scikit-learn으로 정확도 [92%]의 최적화 모델을 도출했습니다.
**[MSA 기반 주문/결제 데이터 시스템 설계]** 대용량 트래픽 환경에서의 데이터 흐름을 이해하기 위해, 마이크로서비스 아키텍처(MSA)를 적용한 이커머스 주문(Order), 결제(Payment), 배송(Delivery) 분산 서버를 직접 구현했습니다. 이를 통해 독립적인 데이터 정합성 유지와 트랜잭션 관리의 중요성을 깨달았습니다.
**[RabbitMQ를 활용한 비동기 데이터 통신]** 분산된 서버 간의 병목 현상을 해결하기 위해 메세지 큐(RabbitMQ) 기반의 비동기 통신 시스템과 OpenFeign을 활용한 REST API를 구축해 보았습니다. 이 경험은 방대한 데이터를 안정적으로 파이프라이닝하고 손실 없이 분석 환경으로 가져오는 데이터 엔지니어링 역량의 밑거름이 되었습니다.
**[웹 애플리케이션 아키텍처 구축]** 더불어 쇼핑몰 팀 프로젝트에 참여하여 실서비스 데이터가 웹 환경에서 어떻게 생성되고 관리되는지 그 생태계를 폭넓게 체득했습니다.

## 3. [성격의 장단점] 문제의 본질을 파고드는 분석력과 협업 기반의 소통력
**[문제의 본질을 파고드는 '원인 분석력']**
복잡한 에러나 논리적 오류를 마주했을 때 단순한 임시 해결이 아니라, 로그와 데이터 흐름을 기반으로 문제의 근본 원인을 추적합니다. 재현 가능한 방식으로 문제를 해결하며, 동일한 오류가 반복되지 않도록 구조적으로 개선하는 데 집중합니다.

**[데이터 신뢰도를 높이는 '정밀한 검증력']**
작은 수치 변화나 이상치도 놓치지 않고 검증하는 습관을 통해 분석 결과의 정확도를 높입니다. 데이터 전처리와 검증 과정을 체계적으로 수행하여, 결과를 그대로 의사결정에 활용할 수 있는 수준의 신뢰도를 확보합니다.

**[결과를 이끌어내는 '협업 기반 소통력']**
단순한 의견 교환을 넘어, 데이터와 근거를 기반으로 소통하며 팀 내 의사결정을 돕습니다. 다양한 관점을 조율하여 최적의 결론을 도출하고, 분석 결과가 실제 실행으로 이어질 수 있도록 연결하는 역할을 수행합니다.

## 4. [협업 및 문제 해결] 신뢰를 바탕으로 한 건강한 충돌과 타협 (크루와의 소통)
최고의 데이터 분석 결과는 결코 개인의 독단으로 이루어질 수 없으며, 동료와의 굳건한 신뢰를 바탕으로 한 치열한 토론과 타협 속에서 탄생한다고 믿습니다. 저는 다양한 사회적 경험을 통해 건강하게 의견을 나누고 최선의 결과를 도출해 내는 탁월한 협업 역량을 길러왔습니다.
일례로 빵집에서 아르바이트를 할 당시, 소수의 인원으로 매장을 관리하며 업무 분담이나 돌발 상황 대처에서 크고 작은 마찰이 발생하곤 했습니다. 저는 갈등을 피하기 위해 무조건 양보하는 것이 아니라, 열린 대화를 주도하여 서로의 업무 고충을 솔직하게 털어놓고 논리적으로 의견을 조율했습니다. 동료에 대한 굳건한 '신뢰'가 있었기에 건강한 충돌이 가능했고, 이를 통해 모두가 만족하는 최선의 업무 프로세스를 타협해 내며 매장 운영의 효율성을 극대화할 수 있었습니다.
또한, 서울시 동행 봉사활동에 꾸준히 참여하며 타인의 입장에 깊이 공감하는 법을 배웠고, 쇼핑몰 구축 팀 프로젝트에서는 여러 차례 발표를 도맡으며 복잡한 기술적 내용을 비전공자도 이해하기 쉽게 전달하는 프레젠테이션 역량을 키웠습니다. 이처럼 신뢰를 바탕으로 솔직하게 소통하며 문제를 해결해 온 저의 경험은, 입사 후 어떠한 부서의 크루(Crew)와도 자유롭게 의견을 나누고 조직 전체의 목표를 향해 시너지를 내는 든든한 밑거름이 될 것입니다.
