[프로젝트 정보]
프로젝트 이름: AI 할 일 추천 앱
목표(한 줄): 사용자의 할 일을 정리하고 우선순위를 자동으로 추천한다
대상 사용자: 대학생, 개발자, 일정 관리가 필요한 사람
[핵심 기능 3개]
1. 할 일 입력: 사용자가 알 일을 텍스트로 입력 가능, 여러개 추가 가능
2. 할 일 환료 체크: 완료버튼 클릭시 체크 표시 완료된 항목은 아래로 이동
3. AI 우선순위 정렬 : 입력된 할 일을 중요도 기준으로 정렬하고 만약에 중요도가 자동으로 되어있으면 AI가 판단해서 중요도 설정
[추가 옵션]
사용 흐름: 할 일 입력 → AI 정렬 → 오늘 할 일 추천 → 완료 체크
성공 기준: 1분 안에 사용 가능, UI직관적으로 해줘, AI 중요도 체크를 자연스럽게 해줘
보조 기능: 알림기능, 간단통계, 다크모드, adminPage
사용자 시나리오: 
1. 사용자가 할 일을 입력한다
2. AI가 우선순위를 정한다
3. 사용자는 추천된 할 일을 수행한다
4. 완료 체크를 한다
